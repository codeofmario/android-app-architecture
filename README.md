# Architecture of an android native app with kotlin

## TECH STACK
- Android
- Kotlin
- Compose
- Lifecycle
- Material
- Paging
- Retrofit
- Coil
- Navigation
- Dagger/Hilt
- Datastore

## START PROJECT
### run docker compose
```console
sudo docker-compose up -d
```

### run seed data
```console
sudo docker exec blogappbackend /bin/sh -c "npm run db:seed"
```
